# Xiaohongshu Diversity Matrix

Use this matrix for Xiaohongshu notes, product seeding posts, carousel image sets, and image-generation prompts.

## Samey Signals

High-risk repetition signals:

- All titles use the same structure, such as "not X, but Y" or "N benefits of X".
- All covers use the same big-title-on-cream-card layout.
- All images are product jar + almonds + spoon on a warm table.
- All posts use breakfast as the only use scene.
- All body copy begins with "recently I started..." or "I changed my breakfast...".
- All selling points list the same nutrients in the same order.
- All CTAs ask the same question.
- All prompts use the same color palette, lighting, and camera angle.

## Minimum Variety For 3 Images

For a 3-image carousel, use at least three different visual mechanics:

| Image role | Must differ by | Example |
|---|---|---|
| Click cover | hook shape, text scale, visual metaphor | bold title + simple illustrated almond object |
| Trust/proof | proof type, camera angle, label system | top-down ingredient map with nutrition labels |
| Use-scene | human action, environment, product role | hand spreading almond butter on toast |

Do not allow all three to share the same background, product placement, prop set, and text overlay style.

## Minimum Variety For 3 Notes

For three Xiaohongshu notes, vary:

- one note can be a myth correction
- one note can be a recipe/use guide
- one note can be a shopping or ingredient checklist

Vary opening styles:

- direct claim: "I finally understand why..."
- scene first: "On rushed mornings..."
- question first: "Why does this feel more filling than sweet spread?"
- mistake avoidance: "Do not buy almond butter only by looking at..."

## Revision Examples

Weak:

- "Almond butter has vitamin E, protein, healthy fat."
- "One spoon in breakfast is satisfying."
- "Add one spoon to toast, oats, fruit."

Stronger differentiation:

- nutrition explainer: "What is inside almond butter?"
- texture proof: "Why it feels richer than sweet spread."
- use guide: "Three ways to finish a busy breakfast."
- shopper angle: "How to choose almond butter without buying a dessert spread."

## Score Interpretation

- `0-7`: revise deeply; the set is template-like.
- `8-12`: acceptable draft, but revise weak items before publishing.
- `13-18`: publishable diversity for a small batch.

For a 3-image set, any score below 12 should trigger regeneration or major prompt revision.
